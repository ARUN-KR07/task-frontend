# Task Frontend - User Management Dashboard

React + Ant Design frontend for User Management Dashboard that connects to a FeathersJS backend.

## Features
- Display list of users in a table
- Add, edit, and soft delete users
- Gender filter (Male / Female / All)
- Loading spinners and success/error messages using Ant Design
- Instant table updates without page refresh

## Tech Stack
- React (Hooks)
- Ant Design
- Axios for API calls
- Connects to FeathersJS REST API

## Prerequisites
- Node.js >= 18
- npm or yarn
- Backend server running and accessible
